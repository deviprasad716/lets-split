# Let's Split 🚗

Let's Split is a community-based ride-sharing platform where drivers and passengers traveling in the same direction can share rides and split fuel costs.

## Vision

Reduce:
- Traffic congestion
- Fuel consumption
- Travel expenses
- Carbon emissions

## Features (Planned)

- User Authentication
- Ride Creation
- Ride Search
- Cost Splitting
- Live Tracking
- Ratings & Reviews
- Emergency Safety Features

Frontend:
- HTML
- CSS
- JavaScript

Future:
